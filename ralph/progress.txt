## 2026-05-09 — BUGFIX: 首页+收藏页白屏 ✅
- **Root cause 1**: `useDidHide` named import from `@tarojs/taro` not properly resolved by api-loader
  - Module 616 (taro.js) only exports Taro API object, no `useDidHide` named export
  - `(0,d.useDidHide)(callback)` → `undefined()` → TypeError → component crash → white screen
  - Fix: removed `useDidHide` from both index.tsx and news.tsx (hooks and usage blocks)
  - 对比: mine.tsx 不使用 useDidHide → 能正常渲染
- **Root cause 2**: `process.env.TARO_APP_API_BASE` not replaced at build time
  - WeChat 小程序可能没有全局 `process` 对象 → ReferenceError → API module init crash
  - Fix: config/index.js `defineConstants` 添加 `'process.env.TARO_APP_API_BASE': JSON.stringify('http://localhost:8001')`
  - 编译后: `const i="http://localhost:8001"` (字符串字面量，无运行时引用)
- **Files changed**:
  - app/src/pages/index/index.tsx (移除 import { useDidHide }, 移除 useDidHide hook)
  - app/src/pages/news/news.tsx (移除 import { useDidHide }, 移除 useDidHide hook)
  - app/config/index.js (defineConstants 添加 TARO_APP_API_BASE)
- **Build**: npx taro build --type weapp → Compiled successfully in 3.26s
- **Note**: `useDidHide` 功能 (切 Tab 停止音频) 暂时损失, 后续可用 page 生命周期或 Taro.eventCenter 替代
