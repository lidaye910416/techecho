## Codebase Patterns
- This codebase uses `src/config/settings.py` for all configuration
- Settings are imported from `src.config.settings` using relative paths with `sys.path.insert`
- Use Python's built-in `sqlite3` for local DB, SQLAlchemy async for MySQL
- All database connections are created/closed per request (no global connection pool for SQLite)
